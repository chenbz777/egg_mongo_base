## 介绍

基于egg基础模版

<br />



## 系统环境

- node => 16.15.0
- npm => 8.5.5



## 快速开始

#### 安装依赖

```bash
# 推荐cnpm
npm install

# yarn
```

#### 运行

```bash
npm run dev

# open http://localhost:7001
```

<br />

## 插件列表

| 插件名称         | 描述            |
| ---------------- | --------------- |
| egg-cors         | 配置跨域        |
| egg-router-group | 路由分组        |
| egg-jwt          | jwt             |
| egg-mongoose     | 连接mongo数据库 |
| egg-validate     | 参数校验        |
